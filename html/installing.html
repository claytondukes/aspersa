Installing Aspersa
Aspersa Manual › Installing Aspersa

Aspersa tools are currently available only by downloading directly from the development trunk. To get a particular tool, execute the following at the command-line:

wget http://aspersa.googlecode.com/svn/trunk/TOOLNAME

Where TOOLNAME is the name of the tool you want to download.

In the future, we plan to release packages for various operating systems to install the tools.
